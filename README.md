# Кошка Груша

У меня есть кошка, ее зовут Груша. 

Вот немного информации о ней:
1. У нее вечно недовольная мордочка
2. Она смешно хрюкает и мяукает
3. Она громко храпит
4. И надо наконец записать ее к грумеру
